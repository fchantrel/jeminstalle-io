# Installation

- S'assurer d'avoir un ``node`` et un ``npm`` sur sa machine
- ``> cd poc-carto``
- ``> npm insall``
- ``> node app.js`` pour ceux qui sont allergiques à coffee, sinon ``> coffee app.coffee``

# Accéder au poc

- http://localhost:3000/map.html
